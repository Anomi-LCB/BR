# Decisions

- [2026-04-03] 외부 API 의존성을 끊고 로컬 JSON과 IndexedDB를 사용하도록 변경함. (Supabase DNS 이슈 회피 목적)
- [2026-04-03] `BibleSearchTab`의 미구현 상태였던 클릭 시 alert을 실제 뷰어인 `FullReadingView`로 대체하여 성경 본문이 정상적으로 조회되도록 조치함. `BibleCard`에도 클릭 이벤트(`onRead`)를 추가하여 데일리 성경 읽기 접근성을 향상시킴.
- [2026-04-03] PDF 추출 정규표현식 `[가-힣]{1,2}\d+:\d+`가 이전 절의 마지막 한글을 탐욕적으로 약어에 포함시키는 문제를 발견. 범용 패턴 대신 각 권의 정확한 약어(BOOK_ABBR 딕셔너리)를 사용하는 방식으로 전환하여 근본적으로 해결. 이 방식은 1글자(창, 출 등) 및 2글자(삼상, 왕하, 고전 등) 약어 모두를 정확히 처리함.
