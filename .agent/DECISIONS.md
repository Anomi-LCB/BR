# Decisions

- [2026-04-03] 외부 API 의존성을 끊고 로컬 JSON과 IndexedDB를 사용하도록 변경함. (Supabase DNS 이슈 회피 목적)
- [2026-04-03] `BibleSearchTab`의 미구현 상태였던 클릭 시 alert을 실제 뷰어인 `FullReadingView`로 대체하여 성경 본문이 정상적으로 조회되도록 조치함. `BibleCard`에도 클릭 이벤트(`onRead`)를 추가하여 데일리 성경 읽기 접근성을 향상시킴.
- [2026-04-03] PDF 추출 정규표현식 `[가-힣]{1,2}\d+:\d+`가 이전 절의 마지막 한글을 탐욕적으로 약어에 포함시키는 문제를 발견. 범용 패턴 대신 각 권의 정확한 약어(BOOK_ABBR 딕셔너리)를 사용하는 방식으로 전환하여 근본적으로 해결. 이 방식은 1글자(창, 출 등) 및 2글자(삼상, 왕하, 고전 등) 약어 모두를 정확히 처리함.
- [2026-04-03] **한글 인코딩(Mojibake) 복구:** 코드 내 하드코딩된 깨진 문자열(`?대떦...`)을 UTF-8 한글로 직접 치환함. 프로젝트 루트의 `fix_encoding.py`와 `dump_group.py`를 참조하여 소실된 UI 텍스트를 전수 복원함.
- [2026-04-03] **빌드 도구 전환(Turbopack → Webpack):** 프로젝트 절대 경로에 한글(`2.교회관련`)이 포함될 경우 Turbopack이 패닉을 일으키며 빌드에 실패함. 임시 방편으로 폴더 이름을 바꾸는 대신 `--webpack` 플래그를 사용하여 Webpack 빌드로 성공함.
- [2026-04-03] **정적 배포 대응:** `output: export` 설정 하에서 `/api` 경로들이 빌드 오류를 유발함. 모든 API 라우트(`chat`, `journal`, `debug-plan`)에 `export const dynamic = 'force-static'`을 추가하여 정적 빌드를 통과시킴.
